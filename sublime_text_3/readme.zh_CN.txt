
使用说明:

1. 将Sublime Text的tar.bz2软件包解压至安装目录中.
	eg.
	```
	sudo tar -xf sublime_text.optimize.tar.bz2 -C /opt/sublime_text
	```
2. 切换到安装目录中, 并运行以sudo用户身份运行./firstrun.sh.
	eg.
	```
	cd /opt/sublime_text
	sudo ./firstrun.sh
	```
3. 启动Sublime Text, 复制已打开的sn.txt中的授权码输入并注册.

反馈及更多: http://sublimetext.iaixue.com
